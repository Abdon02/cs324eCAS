Be sure to have Processing downloaded.

This is the process to run the animation for group_3_assignment4

Steps:
1. Make sure that inside the folder "group_3_assignment4" has the following files:
	"Clouds.pde"
	"Ground.pde"
	"Rain.pde"
	"Suns.pde"
	"UFO.pde"
	"group_3_assignment4.pde"
	"assignment4_description.txt"

2. Open the file, "group_3_assignment4.pde" and hit play

3. Enjoy the animation!

4. Inside the file, "assignment4_description.txt" it will contain the desctiption of the files that were used and created to complete this project.
